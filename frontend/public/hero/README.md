# Hero Images

Place desaturated, low-contrast images here for the film backdrop system.

## Required files
- `field.webp` — foggy field / horizon / soft light (landing hero)
- `constellation.webp` — abstract night sky / bokeh / subtle points (trust page)
- `spiral.webp` — blurred paper / ink / archival texture (how-ai-works)
- `mirror.webp` — soft reflective surface / light spill (pricing)

## Specs
- Format: .webp preferred (fallback to .jpg)
- Size: 1920×1080 minimum
- Style: monochrome or desaturated, low contrast
- The CSS applies opacity ~0.18–0.24 + fog wash + vignette + grain

## Usage
Images are layered behind content with Ken Burns animation (28s cycle).
The grain overlay adds subtle film texture.
All respects `prefers-reduced-motion`.
